# MediHealth Care React Web App

It is a single page application made with ReactJS, React Bootstrap,Route, Firebase CSS etc...

#### Project Featured:

- Load Data from json file and show all Service in UI
- Use Custom Hooks
- Use Route
- Use Private Route
- Firebase Authentication
- Use Context Api
